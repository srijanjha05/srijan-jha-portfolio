My Projects

A showcase of my technical work and case studies.

Click on a project to visit its GitHub repository.

KERNEL_MACOS

A macOS process-tree visualization tool written in C. It programmatically spawns a multi-level process hierarchy while allocating memory and using the vmmap tool to display each process’s memory map, offering insight into memory usage and process relationships.

nextgen-ai

A fun chatbot experiment in Python. It turns text and voice inputs into dynamic conversations by emulating your speech — matching both voice and conversational style — creating a personalized virtual companion experience.

Python_Chat

A desktop chat application built with Python. It combines a multithreaded socket-based server with a Tkinter GUI client to allow real-time, local network messaging. A great demonstration of networking, concurrency, and GUI design.

WEB-CHATBOT

A simple, web-based chatbot developed using Python and Flask. It features a lightweight backend for response logic and a frontend that handles user interaction via the browser.

Snake Eating Game

A classic Snake game built using Python and Pygame. Players control the snake with arrow keys to eat food, grow longer, and rack up points—avoiding walls and self-collision. Features smooth movement, collision detection, and real-time score tracking.

Tic Tac Toe

A straightforward Tic Tac Toe game crafted with Python and Tkinter. It lets two players take turns on a 3×3 grid. The interface includes a “Reset Game” button to quickly start over, showcasing GUI-building and desktop app development fundamentals.

Farmly

An innovative e-commerce platform built on the MERN stack. Farmly empowers farmers by providing a direct-to-consumer marketplace, eliminating middlemen and fostering a transparent connection between producers and buyers.
